# 第3章 組織構造（Organization Structure）

Web3 Proof DAO は、中央管理者を持たない自律分散型組織（DAO：Decentralized Autonomous Organization）として構成される。

主な構成要素は以下の通りである。

1. **メンバー（Members）**  
   DAOの基本単位。証明・提案・投票など、あらゆる活動の主体となる。

2. **コントリビューター（Contributors）**  
   技術・デザイン・法務・広報など、特定領域においてDAOを支える専門参加者。

3. **コアチーム（Core Team）**  
   DAOの維持管理、運営方針の策定、緊急時の技術的サポートを担う中核メンバー。

4. **コミュニティ（Community）**  
   プロジェクト間をつなぎ、DAO全体の活性化と共創を促すネットワーク層。

DAOにおける全ての意思決定は、ガバナンストークンによる投票を基本とし、透明性と公平性をもって実行される。
