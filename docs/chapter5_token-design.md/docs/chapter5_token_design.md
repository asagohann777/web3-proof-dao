# 第5章 トークン設計（Token Design）

DAO運営の透明性・公共性を維持するため、  
トークン分配比率を以下の通りとする。

---

## ● トークン分配比率（設立時）

- 創設者（Founder）：25%  
- 開発パートナー（Developer）  
　（ブロックチェーン簡易登録アプリ10%、ブロックチェーン証明システム10%）：20%  
- DAOトレジャリー（運営資金）：35%  
- 一般参加者（Public）：20%

---

この分配は「証明の民主化（Proof of Democracy）」という理念を実現するための初期設計であり、  
全体の意思決定権が偏らぬよう、将来的なDAO投票制度により調整を行う可能性がある。
