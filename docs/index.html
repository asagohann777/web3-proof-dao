🌏 Web3 Proof DAO（天地愛プロジェクト）🌏

🟦 1. 理念（Philosophy）

与え合う社会の創出。
誰かの創造が、誰かの証明になる。
権威や企業の審査を経ずとも、自らの創作や行動を確かに示せる世界をつくる。
ブロックチェーンによる「証明の民主化」を通じ、信頼が中央から個へと還元される社会を目指す。

🟩 2. 開発経緯と現在（Development History）

  • 2025年5月：「ブロックチェーン簡易登録アプリ」の構想を開始。続いて「ブロックチェーン証明システム」の構想を立案し、個人利用 → 法人利用 → 社会インフラ実装 → メタバース空間への証明技術応用という4段階フェーズ構想を確立。これにより構想全体が成立。
  • 同年5月末： ブロックチェーン簡易登録アプリのプロトタイプ（URLベース）を完成・限定リリース。
  • 同年6月： ブロックチェーン簡易登録アプリおよびブロックチェーン証明システムの実用新案登録申請を実施。現在、補正審査を経て審査中。
  • 現在： DAO内部構成・理念・トークン設計の最終調整を進めつつ、UI図作成構想の深化とオープンチャットによる検証メンバー募集を実施中。また、初期トークン発行（Base／Polygon予定）に向けて準備を進行中。

🟠 3. トークン設計（Tokenomics）

  • 創設者：30％ → 将来的に10％まで段階的縮小（民主化のため）
  • 開発チーム：20％ → 将来的に7.5％まで縮小予定

初期メンバー配布（順次参加順）

  • 001〜100名：1,000トークン
  • 101〜300名：500トークン
  • 301〜500名：200トークン
  • 501〜1,000名：100トークン
  • 1,001名以降：DAO財政状況に応じて少額配布

紹介ボーナス

  • 初期メンバー：紹介1人につき50トークン
  • 以降のメンバー：紹介1人につき30トークン

🟧 4. DAO構成（DAO Structure）

  • ガバナンス方式：Proof-based Decision
  • 投票権限：トークン保有量に比例（1Token = 1Vote）
  • 議題提出権：500Token以上の保持者
  • 公開範囲：理念・議題・投票結果は全公開
  • 非公開領域：開発中コード・実用新案関係資料
