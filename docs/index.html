🌏Web3 Proof DAO（天地愛プロジェクト）🌏

🟦 1. 理念（Philosophy）

与え合う社会の創出。
誰かの創造が、誰かの証明になる。
権威や企業の審査を経ずとも、自らの創作や行動を確かに示せる世界をつくる。
ブロックチェーンによる「証明の民主化」を通じ、信頼が中央から個へと還元される社会を目指す。

🟩 2. 開発経緯と現在（Development History）

  • ■2025年5月：「ブロックチェーン簡易登録アプリ」の構想を開始。
  • ■同年5月末：プロトタイプ（URLベース）を完成・限定リリース。
  • ■同年6月：「ブロックチェーン簡易登録アプリ」および「ブロックチェーン証明システム」を実用新案登録申請。現在審査中。
  • ■現在：DAO内部構成・理念・トークン設計の最終調整を進行中。UI図作成構想や検証メンバー募集も実施中。

🟠 3. トークン設計（Tokenomics）

  • 創設者：30％ → 将来的に10％まで縮小
  • 開発チーム：20％ → 将来的に7.5％まで縮小
  • 001〜100名：1,000トークン／101〜300名：500トークン／301〜500名：200トークン／501〜1,000名：100トークン
  • 紹介ボーナス：初期メンバー1人紹介で50トークン／以降は30トークン

🟧 4. DAO構成（DAO Structure）

  • ガバナンス方式：Proof-based Decision
  • 投票権：トークン保有量に比例（1Token = 1Vote）
  • 議題提出権：500Token以上保持者
  • 理念・議題・投票結果は全公開／コード・実用新案資料は非公開
